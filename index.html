Responsive Burger Menu Template with CSS

What is a hamburger menu?

A hamburger menu is an icon used on a website and in apps that, when clicked or tapped, opens a side menu or navigation drawer. It’s called a “hamburger menu” because it takes the form of the famous sandwich. This icon was created by the designer Norm Cox for the graphical user interface of the Xerox Star workstation in 1981. Cox’s goal was to create an icon that would communicate to users that a list of items was hidden behind it.

Usage

In styles directory, the menu.css file will be the backbone, then we can comnbine with with-css.css to implement in pure CSS or with-js.css, combined with js/menu.js to implement with javascript. The different is, with CSS, the menu will be displayed via focus event. In the javascript version, onclick() event is handled.

  • In pure css mode, the burger menu will be shown up when the nav bar area is hover or focused.
  • By implementing javascript, the menu is triggered by click action as usual.

The mobile.css should be added to support mobile devices.
